B.E Project

Our objective was to create a blockchain technology-based web-application that will act as an alternative to the traditional Authentication scheme for IoT Devices. The model is to provide enhanced security for IoT devices using Blockchain Technology.This is to provide a secure and decentralized system that can verify the identity of connected devices and ensure that only authorized devices can access the network.By using blockchain technology, it is possible to create a temper proof and transparent ledger of device identities and their associated permissions. With blockchain based authentication, there is no single point of failure or vulnerability in the system, as the ledger is distributed across a network nodes.

Technologies used: Node.js, Express, HTML,CSS, Javascript and Ether.js
